Flexbox

The Flexible layout module (or Flexbox) is a one-dimensional layout module for distributing space between items. In this context, One-dimensional refers to Flexbox arrainging items in either a row or column. This is contrasted with the two-dimensional layout of 'CSS Grid Layout', which controls columns and rows together.

The Flex Container

Elements that arrange their children using the Flexbox layout are called flex containers. To create a flex container, set the element's display property to flex. For example, to use Flexbox on a div use

<div style='display: flex;'><div>
div {
  display: flex;
}
